# DeAI-zh：中文去AI化重写 Skill

## 解决什么问题

AI 生成的中文文本有明显的"机器味"：排比句堆砌、"随着...的不断发展"万能开头、"不仅...更..."公式化结构、"值得注意的是"等连接词泛滥、空洞的未来展望结尾。这些模式让文章一眼就能被识别为 AI 生成，降低了内容的可信度和阅读体验。

DeAI-zh 针对中文语境，系统梳理了 24 种常见的 AI 写作模式（分为内容、语言、风格、交流四大类），并提供具体的重写策略和 before/after 示例，帮助将 AI 生成的文本改写为自然、有观点、有温度的人类风格文本。

与英文版 humanizer 的区别：
- 去掉了中文里不适用的英文模式（如 Title Case、系动词回避）
- 补充了中文特有的 AI 痕迹（万能开头公式、AI 高频词汇表、三段式排比等）
- 提供中文 before/after 对比示例
- 支持按场景（技术博客 / 社交媒体）调整重写策略

## Agent 如何使用

**触发方式：**

在 Claude Code 中输入 `/deai-zh` 激活，或在对话中自然触发（当用户提到"去AI化"、"太像AI写的"、"让文字更自然"、"人性化改写"、"改得像人写的"等表述时自动触发）。

**使用示例：**

```
/deai-zh 帮我重写这段技术博客：[粘贴文本]
```

```
这段公众号文章太AI了，帮我改自然点：[粘贴文本]
```

```
/deai-zh 处理一下 article.md 里的内容，场景是社交媒体
```

**处理流程：**

1. Agent 读取 SKILL.md 中的 24 种 AI 模式清单和重写原则
2. 判断文本场景（技术博客 or 社交媒体），选择对应策略
3. 逐段识别 AI 痕迹并重写
4. 按质量检查标准（直接性、节奏感、人味、立场、简洁）自检
5. 输出重写结果

## 输出什么结果

直接输出重写后的完整文本，不附带修改说明或标注。

**输出特征：**

- 保留原文核心信息和论点，只改表达方式
- 去除 AI 高频词汇和公式化结构
- 注入个人观点和立场（不再两面讨论什么都没说）
- 节奏自然（长短句交替，段落长度不均匀）
- 语气匹配场景（技术博客稍正式但不学术，社交媒体像朋友聊天）
- 技术术语保持原样，不牺牲准确性
